  <action>
Create an animated component that displays floating +XP text after set completion.

**Props:**
```typescript
interface XPFloaterProps {
  xp: number;           // XP amount to display
  visible: boolean;     // Controls animation trigger
  onComplete: () => void; // Called when animation finishes
}
```

**Animation behavior:**
1. When visible becomes true:
   - Text starts at opacity 1, translateY 0
   - Animates upward (translateY -60) over 800ms
   - Fades out (opacity 0) during last 400ms
2. After animation completes, call onComplete
3. Display format: "+{xp} XP" (e.g., "+35 XP")

**Implementation:**
- Use react-native-reanimated useSharedValue, useAnimatedStyle, withTiming, withSequence, runOnJS
- Use Reanimated.View for animated container
- Style: ember-orange color, fonts.mono, fontSize 24, positioned absolute center-top of parent
- useEffect watches visible prop to trigger animation
- On animation complete, use runOnJS to call onComplete from worklet

**Example animation sequence:**
```typescript
translateY.value = withTiming(-60, { duration: 800 });
opacity.value = withSequence(
  withTiming(1, { duration: 400 }),
  withTiming(0, { duration: 400 })
);
```
