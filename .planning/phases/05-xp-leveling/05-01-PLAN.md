```typescript
export interface SetXPInput {
  weightKg: number;
  reps: number;
  tempoEnabled: boolean;
  isPR: boolean;
  applyConsistencyBonus: boolean;  // Caller controls when to apply (once per session)
}

export interface XPResult {
  base: number;           // Always 10
  volumeBonus: number;    // floor(weight * reps / 100)
  prBonus: number;        // 50 if isPR, else 0
  consistencyBonus: number; // 20 if applyConsistencyBonus, else 0
  tempoMultiplier: number;  // 1.5 if tempo, else 1.0
  subtotal: number;         // base + volumeBonus + prBonus + consistencyBonus
  total: number;            // floor(subtotal * tempoMultiplier)
}
```

**Functions to implement:**

1. `calculateSetXP(input: SetXPInput): XPResult`
   - base = 10 (always)
   - volumeBonus = Math.floor(weightKg * reps / 100)
   - prBonus = isPR ? 50 : 0
   - consistencyBonus = applyConsistencyBonus ? 20 : 0  // Caller decides when to apply
   - tempoMultiplier = tempoEnabled ? 1.5 : 1.0
   - subtotal = base + volumeBonus + prBonus + consistencyBonus
   - total = Math.floor(subtotal * tempoMultiplier)
   - Return all components for breakdown display

2. `getLevelFromXP(totalXp: number): number`
   - Import LEVEL_THRESHOLDS from @/constants/xp
   - Find highest level where totalXp >= xpRequired
   - Return level (1-10)

3. `getXPProgress(totalXp: number, level: number): { current: number; needed: number; progress: number }`
   - current = XP earned toward next level
   - needed = XP required to reach next level
   - progress = current / needed (0-1)
   - If max level, return { current: 0, needed: 0, progress: 1 }

4. `wasTrainedWithin7Days(lastTrainedAt: string | null): boolean`
   - Return false if null
   - Compare lastTrainedAt to now with 7-day (604800000ms) threshold

**Implementation notes:**
- Use @/constants/xp for LEVEL_THRESHOLDS
- All math uses Math.floor for integer XP
- No side effects, no state
- Consistency bonus is controlled by caller, not by this function checking dates

Create a React hook that provides database operations for XP persistence.

**Hook interface:**
```typescript
export function useXPService() {
  return {
    checkIsPR,          // Query historical max weight for exercise
    getZoneConsistency, // Check if zone was trained in last 7 days
    awardSetXP,         // Calculate XP (does NOT write to DB - accumulation happens in session)
    finalizeSession     // Write zone_stats after session complete
  };
}
```

**Functions to implement:**

1. `checkIsPR(exerciseId: string, weightKg: number, reps: number): Promise<boolean>`
   - Query workout_sets: `SELECT MAX(weight_kg) as max_weight FROM workout_sets WHERE exercise_id = ? AND reps >= ?`
   - Return true if no prior sets OR weightKg > max_weight
   - This determines if +50 PR bonus applies

2. `getZoneConsistency(zoneId: string): Promise<boolean>`
   - Query zone_stats: `SELECT last_trained_at FROM zone_stats WHERE zone_id = ?`
   - Use wasTrainedWithin7Days from xp-calculator
   - Return true if trained within 7 days (consistency bonus eligible)
   - Caller decides whether to actually apply it (once per session control)

3. `awardSetXP(params: { zoneId: string, exerciseId: string, weightKg: number, reps: number, tempoEnabled: boolean, sessionId: string, applyConsistencyBonus: boolean }): Promise<XPResult>`
   - Call checkIsPR to determine isPR
   - Call calculateSetXP with all parameters including applyConsistencyBonus from caller
   - Return the XPResult (caller accumulates and uses for UI display)
   - NOTE: Does NOT write to database - that happens in finalizeSession

4. `finalizeSession(zoneId: string, totalXP: number, sessionId: string): Promise<{ leveledUp: boolean; newLevel: number }>`
   - Use withExclusiveTransactionAsync for atomic multi-table update
   - Read current total_xp, level from zone_stats
   - Calculate new total_xp = current + totalXP
   - Calculate new level using getLevelFromXP
   - Determine if leveled up (newLevel > currentLevel)
   - Update zone_stats: total_xp, level, last_trained_at = datetime('now'), current_streak (increment if was warm, else reset to 1)
   - Insert xp_history record with source = 'SESSION', source_id = sessionId
   - Return { leveledUp, newLevel }

**Implementation notes:**
- Import from useSQLiteContext, xp-calculator
- Use generateId pattern from useWorkoutSession for xp_history.id
- Transaction ensures zone_stats + xp_history update atomically
- Streak logic: if wasTrainedWithin7Days, increment current_streak; else reset to 1
- Consistency bonus timing controlled by caller via applyConsistencyBonus param
