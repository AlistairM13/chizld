**useElapsedTimer.ts:** Create a new hook that counts up from 0 in whole seconds.

Interface:
```typescript
export function useElapsedTimer(): {
  elapsedSeconds: number;
  formatted: string;  // "Xm Xs" format for < 1h, "Xh Ym" for >= 1h
  start: () => void;
  stop: () => void;
  isRunning: boolean;
}
```

Implementation:
- Use `useRef` for start timestamp (Date.now()) and interval ID
- `setInterval` at 1000ms updating a `useState<number>` for elapsed seconds
- `start()` records `Date.now()` in ref, begins interval
- `stop()` clears interval
- `formatted` derives from elapsedSeconds: under 60 = "Xs", under 3600 = "Xm Ys", else "Xh Ym"
- Clean up interval on unmount
- Auto-start on mount (call start in useEffect with empty deps)

**Layout (landscape-friendly):**

- Track: horizontal bar, full width, height 4px, colors.zone.cold background
- Fill: left portion of track up to current value position, colors.ember[500]
- Thumb: 28x28 circle at current value position, colors.ember[500] with white border
- Labels: RPE numbers 6, 7, 8, 9, 10 evenly spaced below the track, fonts.mono size 10, colors.text.muted (selected value in colors.ember[500])

**Gesture:**
- Use `Gesture.Pan()` from `react-native-gesture-handler` v2 API (NOT legacy API)
- Wrap thumb+track in `GestureDetector`
- On pan update: calculate position along track width, clamp to 0-1, map to RPE 6-10 (snap to nearest integer)
- On pan end: fire `onChange` with snapped value, trigger `Haptics.selectionAsync()`
- Also support tap on track to jump to value

**Visual pulse on change:**
- Use Reanimated `withSequence(withTiming(1.15, {duration: 80}), withTiming(1, {duration: 80}))` on thumb scale when value changes
- Haptic feedback on every value snap (Haptics.selectionAsync)

**Important:** Use `Gesture.Pan()` and `GestureDetector` (v2 API), NOT `PanGestureHandler` (legacy API). The v2 API works with GestureHandlerRootView.

Use `useSharedValue` for the gesture X position, `useAnimatedStyle` for thumb position, `runOnJS` to call onChange from the gesture handler worklet.
