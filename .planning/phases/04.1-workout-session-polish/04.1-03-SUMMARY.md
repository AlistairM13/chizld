## Decisions Made
- `isComplete` state gates the completion flow: when timer hits zero, set state true + haptic warning, do NOT call `onComplete()` automatically. User taps the full overlay Pressable to dismiss. This prevents jarring auto-navigation mid-rest.
- Background is `'#000000'` (not `colors.bg.primary`) — pure black is intentional for visual contrast.
- Raw seconds displayed (e.g., `90` not `1:30`) per plan spec — simpler and more visceral countdown.
