**Visual pulse on weight/reps input change:**

Add a Reanimated shared value for input pulse:

```typescript

}));
```

When weight or reps changes, trigger pulse:
```typescript
// In the weight onChange handler:
const handleWeightChange = (v: number) => {
  setCurrentSet((s) => ({ ...s, weightKg: v }));

    withTiming(1.05, { duration: 80 }),

  );

  setCurrentSet((s) => ({ ...s, reps: v }));

    withTiming(1.05, { duration: 80 }),
    withTiming(1, { duration: 80 })
  );
};
```

Wrap the bottom input row in `<Animated.View style={[styles.bottomInputRow, inputPulseStyle]}>`.
