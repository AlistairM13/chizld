    Enhance CharacterScreen to manage detail mode state and drive all animations from a single shared value with staggered choreography (character slides FIRST, then stat card enters after delay):

    1. Add imports from react-native-reanimated:
       - useSharedValue, useAnimatedStyle, withTiming, withDelay, interpolate, Easing, useDerivedValue

    2. Create animation drivers with staggered timing:
       ```typescript
       const detailProgress = useSharedValue(0);
       const statCardProgress = useSharedValue(0);

       useEffect(() => {
         if (selectedZone !== null) {
           // Character slides first
           detailProgress.value = withTiming(1, {
             duration: 400,
             easing: Easing.out(Easing.cubic),
           });
           // Stat card enters after ~100ms delay
           statCardProgress.value = withDelay(100, withTiming(1, {
             duration: 400,
             easing: Easing.out(Easing.cubic),
           }));
         } else {
           // Reverse: stat card exits first, then character slides back
           statCardProgress.value = withTiming(0, {
             duration: 300,
             easing: Easing.in(Easing.cubic),
           });
           detailProgress.value = withDelay(100, withTiming(0, {
             duration: 400,
             easing: Easing.out(Easing.cubic),
           }));
         }
       }, [selectedZone]);
       ```

    3. Create animated styles for character offset (~50% left per user decision):
       ```typescript
       const characterAnimatedStyle = useAnimatedStyle(() => ({
         transform: [
           { translateX: interpolate(detailProgress.value, [0, 1], [0, -width * 0.5]) }
         ],
       }));
       ```

    4. Create animated style for zone card opacity (fade out in detail mode):
       ```typescript
       const cardOpacity = useDerivedValue(() =>
         interpolate(detailProgress.value, [0, 0.3], [1, 0])
       );
       ```

    5. Pass detailProgress to BodyCanvas (for character offset) and to each ZoneCard (for fade).
       Pass statCardProgress to StatCard (will be wired in Plan 02).

    6. Wrap the BodyCanvas container in Animated.View with characterAnimatedStyle.

    IMPORTANT: Use Extrapolation.CLAMP for all interpolations to prevent overshoot. Import Extrapolation from reanimated.
    IMPORTANT: The choreography is intentional - character slides FIRST, then stat card enters after ~100ms delay per user decision.
