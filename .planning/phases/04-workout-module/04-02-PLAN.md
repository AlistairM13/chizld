Create useWorkoutSession hook (src/hooks/useWorkoutSession.ts):
- Use useSQLiteContext() for database access
- State: sessionId, exercises (array of { id, name, sets: SetData[] }), currentExerciseIndex
- SetData type: { setNumber: number; weightKg: number; reps: number; rpe: number; completedAt: string }
- Functions:
  - createSession(exerciseIds: string[]): Creates workout_sessions row, returns sessionId, loads exercise names
  - addSet(exerciseId: string, data: { weightKg, reps, rpe }): INSERT INTO workout_sets, returns set ID
  - completeSession(notes?: string): UPDATE workout_sessions SET ended_at
  - getSessionSummary(): Returns { totalSets, exercisesCompleted, exercises array with set counts }
- Generate IDs: Date.now().toString(36) + Math.random().toString(36).substr(2)

Create useRestTimer hook (src/hooks/useRestTimer.ts):
- State: remaining (seconds), isRunning, duration (default 90)
- Use useRef for endTimeRef to track absolute end time (avoids drift)
- Functions:
  - start(customDuration?: number): Sets endTimeRef, starts interval
  - skip(): Stops timer immediately
  - adjustTime(delta: number): Adds delta to endTimeRef, updates remaining
  - setDefaultDuration(seconds: number): Updates default
- useEffect with setInterval (100ms) to update remaining from endTimeRef
- Clean up interval on unmount
- Return { remaining, isRunning, start, skip, adjustTime, setDefaultDuration }
  </action>

  </verify>
