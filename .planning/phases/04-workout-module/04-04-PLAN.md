Create useSessionSummary hook (src/hooks/useSessionSummary.ts):

Accept sessionId parameter.

Query workout_sessions and workout_sets to aggregate:
- Session start/end times
- Total sets count: COUNT(*) from workout_sets WHERE session_id = ?
- Exercises with sets: Group workout_sets by exercise_id, join with exercises for names
- Total volume: SUM(weight_kg * reps) from workout_sets

Return interface SessionSummary:
- sessionId: string
- startedAt: Date
- endedAt: Date | null
- duration: number (minutes)
- totalSets: number
- exercises: Array<{ id: string; name: string; setCount: number }>
- totalVolume: number (kg)
- xpEarned: number (PLACEHOLDER: Calculate as totalSets * 10 for now - Phase 5 will replace)

Use useSQLiteContext() for database access.
Use useEffect to fetch on mount.

Return { summary: SessionSummary | null, isLoading: boolean }
